# Grocery Automation
## Goals and Purpose
Like a "responsible person", I spend a chunk of time each week planning out meals and making a grocery list. I currently map out the week in Google Sheets in order to collaborate with my partner, and create a grocery list based on the planned meals. Before this project, I would manually copy over the grocery list from the spreadsheet to AnyList, a nifty app for list making that lives on mobile. 

As part of a **#AutomateTheBoringStuff** initiative, I dug into the Google Sheets API to see how, with a script a single command in Terminal, I could push my grocery needs to AnyList. This project includes a second dimension for each list item ("category"), which is a way AnyList can group items based on their location in the store for a more efficient shopping experience. 

The specific project goals: 
  * Eliminate need for manual data entry on mobile app
  * Simplify list export process within Google Sheets for decreased interaction
  * Initiate push to app with simple command in Terminal
